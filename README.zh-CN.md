# 反应速度测试

[![English](https://img.shields.io/badge/language-English-blue.svg)](README.md)

一个用于测量您反应速度的简单网页应用。

## 工作原理

该测试会测量您对视觉刺激做出反应所需的时间。

1.  屏幕上会显示 “点击开始” 的提示。
2.  点击后，屏幕将变为红色并显示 “等待变绿...”。
3.  当屏幕变为绿色时，请尽快点击！
4.  您的反应时间将以毫秒为单位显示出来。

## 如何使用

1.  克隆此仓库。
2.  在您的网页浏览器中打开 `index.html` 文件。

就这么简单！
